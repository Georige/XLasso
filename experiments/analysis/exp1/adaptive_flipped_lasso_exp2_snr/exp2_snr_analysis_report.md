## 0. 框架使用说明

### SNR研究启动方式

```bash
# 使用sweep.py的snr模式
python factory/sweep.py snr --config configs/snr/adaptive_flipped_lasso_exp2_snr.yaml

# 可用模式:
# - stage1: 参数网格搜索
# - stage2: 细粒度CV调参
# - snr:    信噪比敏感性分析
```

### SNR配置文件格式

```yaml
experiment: adaptive_flipped_lasso_exp2_snr
algo: adaptive_flipped_lasso

# 数据生成参数
n_samples: 300
n_features: 500
n_nonzero: 20
correlation_type: ar1
rho: 0.8

# 固定参数 (来自stage1最优配置)
lambda_ridge: 1.0
lambda_: 1.0
gamma: 0.5

# SNR研究设置
n_repeats: 5
cv_folds: 1

# SNR网格 (sigma为噪声标准差, SNR=1/sigma)
search_space:
  sigma: [0.1, 0.3, 0.5, 0.7, 1.0, 1.5, 2.0, 3.0]

output_dir: experiments/results/snr
```

### 输出文件

- `raw.csv`: 所有重复实验的原始数据
- `summary.csv`: 按sigma分组的汇总统计
- `configs/snr/{experiment}_best.yaml`: 最优配置和敏感性分析

## 1. 实验概述

本实验在Exp2数据配置下，研究不同信噪比(SNR)对AdaptiveFlippedLasso模型性能的影响。Exp2采用AR(1)相关性结构（rho=0.8），前20个奇数索引变量（1,3,5,...,39）的系数为1.0，其余为0。

### 1.1 实验设置

- **数据生成**: AR(1)相关性结构, rho=0.8
- **样本量**: n=300
- **特征数**: p=500
- **非零系数数**: 20
- **信噪比范围**: sigma = [0.1, 0.3, 0.5, 0.7, 1.0, 1.5, 2.0, 3.0] (对应SNR = 10.0, 3.33, 2.0, 1.43, 1.0, 0.67, 0.5, 0.33)
- **固定参数**: lambda_ridge=1.0, lambda_=1.0, gamma=0.5 (来自Exp2 Stage1最优配置)

### 2.2 SNR 敏感性分析

| 指标 | 与sigma的相关性 | 显著性 | 解读 |
|------|----------------|--------|------|
| F1 | r = -0.91 | 强负相关 | sigma增大 → F1显著下降 |
| MSE | r = +0.96 | 强正相关 | sigma增大 → MSE显著上升 |
| FDR | r = +0.91 | 强正相关 | sigma增大 → 假阳性率上升 |
| TPR | r = -0.84 | 强负相关 | sigma增大 → 召回率下降 |
| R2 | r = -0.97 | 强负相关 | sigma增大 → R2显著下降 |

### 2.3 关键发现

1. **高SNR区域 (σ ≤ 0.5, SNR ≥ 2.0)**:
   - F1 > 0.97, 性能优异且稳定
   - TPR = 1.00, 完全检测到所有真信号
   - FDR < 0.06, 假阳性率极低
   - 方差小，实验重复性好

2. **中等SNR区域 (σ = 1.0, SNR = 1.0)**:
   - F1 ≈ 0.93, 性能可接受
   - TPR仍保持1.00
   - FDR上升至0.13
   - 模型仍能有效选择特征

3. **低SNR区域 (σ ≥ 2.0, SNR ≤ 0.5)**:
   - F1显著下降 (最低0.67)
   - TPR下降至0.78, 漏检开始出现
   - FDR急剧上升至0.41, 假阳性严重
   - 方差增大，稳定性下降

## 3. 参数影响分析

### 3.1 Stage1网格搜索参数敏感性

基于Exp2 Stage1网格搜索结果（36个参数组合 × 5次重复）:

| 参数 | 与F1相关性 | 影响方向 | 最优范围 |
|------|-----------|---------|---------|
| **lambda_** | r = +0.55 | 正相关 | [0.5, 1.0] |
| **gamma** | r = -0.65 | 负相关 | [0.5, 1.0] |
| **lambda_ridge** | r ≈ 0 | 弱影响 | [1.0, 50.0] |

### 3.2 各参数对指标的影响

#### lambda_ (Lasso正则化强度)

| lambda_ | F1 (mean±std) | FDR (mean±std) | MSE (mean±std) |
|---------|---------------|----------------|----------------|
| 0.01 | 0.15±0.04 | 0.92±0.02 | 4.84±3.68 |
| 0.10 | 0.26±0.12 | 0.88±0.04 | 3.62±2.24 |
| 0.50 | 0.49±0.31 | 0.64±0.30 | 2.93±1.77 |
| 1.00 | 0.56±0.33 | 0.51±0.33 | 2.92±1.89 |

**解读**: lambda_ 越大，F1越高，FDR越低。lambda_控制Lasso惩罚力度，较大的值促进更稀疏的解。

#### gamma (权重衰减参数)

| gamma | F1 (mean±std) | FDR (mean±std) | MSE (mean±std) |
|-------|---------------|----------------|----------------|
| 0.5 | 0.60±0.31 | 0.44±0.33 | 2.21±1.51 |
| 1.0 | 0.36±0.21 | 0.78±0.13 | 3.51±1.97 |
| 2.0 | 0.13±0.03 | 0.94±0.02 | 6.53±4.14 |

**解读**: gamma 越大，F1越低，FDR越高。gamma控制权重计算的衰减速度，较小的gamma有利于特征选择。

#### lambda_ridge (岭回归正则化)

| lambda_ridge | F1 (mean±std) | FDR (mean±std) |
|--------------|---------------|----------------|
| 1.0 | 0.38±0.30 | 0.67±0.29 |
| 10.0 | 0.36±0.29 | 0.69±0.28 |
| 50.0 | 0.35±0.28 | 0.71±0.27 |

**解读**: lambda_ridge对性能影响较小，在[1.0, 50.0]范围内无显著差异。

### 3.3 最优配置

| 参数 | 最优值 |
|------|--------|
| lambda_ridge | 1.0 |
| lambda_ | 1.0 |
| gamma | 0.5 |

**最优性能**: F1 = 0.9524, MSE = 2.3026, TPR = 1.0000, FDR = 0.0909

## 4. 结论与建议

### 4.1 SNR影响总结

1. AdaptiveFlippedLasso在高SNR条件下表现优异（F1>0.97）
2. 随着噪声增加，模型性能逐渐下降，主要体现在FDR上升和TPR下降
3. 在极低SNR（σ≥3.0）时，模型性能严重下降，不建议使用

### 4.2 参数调优建议

1. **lambda_**: 建议设置在[0.5, 1.0]范围内，越大越好
2. **gamma**: 建议设置在[0.5, 1.0]范围内，越小越好
3. **lambda_ridge**: 可在较宽范围[1.0, 50.0]内选择，影响较小
