# Lasso算法基准测试报告

**实验日期**: 2026-03-26
**场景**: Exp1 (n=300, p=500, pairwise ρ=0.5, 前20个变量 β=1.0)
**重复次数**: 10次 × 6个sigma值 = 60次实验/算法
**验证方法**: AdaptiveFlippedLasso使用单次80/20分割，其他算法使用5折交叉验证(CV=5)

---

## 1. 实验配置

| 参数 | 值 |
|------|-----|
| n_samples | 300 |
| n_features | 500 |
| n_nonzero (真实稀疏度) | 20 |
| correlation_type | pairwise |
| rho | 0.5 |
| sigma范围 | [0.1, 0.5, 1.0, 1.5, 2.0, 3.0] |
| n_repeats | 10 |
| AdaptiveFlippedLasso | cv_folds=1 (单次分割) |
| 其他算法 | cv_folds=5 (5折CV) |

**测试算法**:
1. **AdaptiveFlippedLasso** (v3.6.0) - 修复后的自适应翻转Lasso
2. **UniLasso** - 单变量引导稀疏回归
3. **AdaptiveLasso** - 自适应Lasso
4. **GroupLasso** - 组Lasso
5. **FusedLasso** - 融合Lasso
6. **AdaptiveSparseGroupLasso** - 自适应稀疏组Lasso

## 8. 结论

### 8.1 算法排名 (按F1性能)

1. **AdaptiveFlippedLasso** - 全面领先，高SNR下F1接近1.0
2. **AdaptiveSparseGroupLasso** - 中等性能，特征选择过于保守
3. **AdaptiveLasso** - 召回率极低，严重欠拟合
4. **UniLasso** - 召回率高但FDR极高
5. **GroupLasso** - 大量假阳性
6. **FusedLasso** - 表现最差，不稳定

### 8.2 适用场景建议

| 算法 | 推荐场景 |
|------|---------|
| AdaptiveFlippedLasso | 高/中SNR稀疏特征选择，生物标志物筛选 |
| AdaptiveSparseGroupLasso | 需要保守选择、低FPR的应用 |
| AdaptiveLasso | 不推荐（欠拟合严重） |
| UniLasso/GroupLasso/FusedLasso | 不推荐（高FDR） |

### 8.3 AdaptiveFlippedLasso核心优势

1. **超高F1**: 0.998 (高SNR) → 0.58 (低SNR)，全面领先
2. **极低MSE**: 比其他算法低100倍以上
3. **精确特征选择**: n_selected≈20 (接近真实稀疏度)
4. **高稳定性**: CV<3% (高SNR条件下)
5. **完美召回**: σ≤2.0时TPR=100%
