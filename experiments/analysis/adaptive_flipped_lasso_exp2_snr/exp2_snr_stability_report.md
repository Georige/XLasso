## 4. 关键发现

### 4.1 高SNR区域 (σ ≤ 0.5)
- F1 > 0.97, CV% < 2.5%
- TPR = 1.00 (完全检测所有真信号)
- FDR < 0.05 (假阳性率极低)
- 模型性能优异且高度稳定

### 4.2 中等SNR区域 (σ = 0.7 ~ 1.0)
- F1 0.94-0.96, CV% 2.5-4%
- TPR = 1.00 (保持完全检测)
- FDR 0.08-0.11 (可接受的假阳性率)
- 模型稳定，适合常规使用

### 4.3 低SNR区域 (σ ≥ 1.5)
- F1 < 0.91, CV% > 5%
- TPR开始下降 (最低0.85)
- FDR显著上升 (最高0.38)
- 变异性增加，需谨慎解释结果

### 4.4 指标稳定性排序 (按CV%)
- 最稳定: TPR, Recall, Sparsity (CV% < 5%)
- 中等稳定: F1, Precision, R2 (CV% 5-20%)
- 变异较大: MSE, FDR (CV% > 30%)
