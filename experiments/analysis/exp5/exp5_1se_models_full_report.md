# Exp5 绝对隐身陷阱场景 - 各模型1-SE版本全面对比报告

## 1. 实验概述

### 1.1 实验目的
评估在**绝对隐身陷阱（Perfect Masking / Absolute Invisible Trap）**场景下，主流稀疏变量选择模型的1-SE版本的表现。该场景是论文中设计的"魔鬼等级"挑战，核心难度在于通过精确设计X与y的关系，使真实信号的**边际相关性归零**，从而使依赖边际相关的传统变量选择方法失效。

### 1.2 数据配置

| 参数 | 值 | 说明 |
|------|-----|------|
| n_samples | 300 | 样本数 |
| n_features | 1000 | 特征数 |
| n_nonzero | 20 | 真实非零系数（10对孪生变量） |
| correlation_type | experiment5 | 绝对隐身陷阱专用结构 |
| rho | 0.8 | 孪生变量对内相关性 |
| sigma | 1.0 | 噪声标准差 |
| SNR | 1.0 | 信噪比 |
| n_repeats | 5 | 重复次数 |

### 1.3 真实信号结构

- **10对孪生变量**：β_{2t-1}=2.0, β_{2t}=-2.5 (t=1,...,10)
- **共20个真实非零系数**
- **核心挑战**：通过Householder变换使每个孪生变量对的边际相关性归零

### 1.4 对比模型

| 模型 | 全称 | 1-SE规则 | 特点 |
|------|------|----------|------|
| UniLassoCV_1SE | Univariate-Guided Lasso CV | 是 | 利用单变量回归指导 |
| LassoCV_1SE | sklearn Lasso CV | 是 | 标准套索交叉验证 |
| AFLCV_1SE | Adaptive Flipped Lasso CV | 是 | 自适应翻转套索 |
| RelaxedLassoCV_1SE | Relaxed Lasso CV | 是 | Lasso后接OLS去偏 |
| ElasticNetCV_1SE | Elastic Net CV | 是 | L1+L2混合惩罚 |
| AdaptiveLassoCV_1SE | Adaptive Lasso CV | 是 | 自适应权重Lasso |

## 4. 关键发现

### 4.1 绝对隐身陷阱的杀伤力

**所有模型的F1都接近于0**，TPR几乎为0，这验证了绝对隐身陷阱场景的极端难度：

1. **边际相关性归零**：真实信号的边际相关系数被设计为接近0
2. **传统方法失效**：依赖边际相关的Lasso、Adaptive Lasso等完全失效
3. **1-SE过于保守**：1-SE规则在此场景下将模型压缩到近乎零选择

### 4.2 模型分层表现

#### 第一层：相对最佳（LassoCV_1SE）
- **F1=0.0016**，略高于其他模型
- **n_selected=7.9**，有一定选择但仅约2% TPR
- **FDR=0.3186**，选中的特征中约32%是假阳性
- 表现最好的原因可能是：Lasso对弱信号有一定的随机捕获能力

#### 第二层：过度选择（UniLassoCV_1SE）
- **F1=0.0015**，与LassoCV_1SE相当
- **n_selected=13.6**，比LassoCV_1SE多选择约72%
- **FDR=0.7588**，选中特征中约76%是假阳性
- UniLasso的单变量指导在此场景下产生了更多的假阳性

#### 第三层：完全失效（4个模型）
- **AFLCV_1SE、ElasticNetCV_1SE、AdaptiveLassoCV_1SE**：n_selected=0
- **RelaxedLassoCV_1SE**：n_selected=0.6（几乎为零）
- 这些模型的1-SE规则过于保守，完全无法选出任何特征

### 4.3 预测性能 vs 变量选择性能的矛盾

## 5. 场景分析

### 5.1 绝对隐身陷阱的设计原理

```python
# 对每个孪生变量对施以与y精确抵消的调整
v1 = X[:, 2*i].copy()
v2 = X[:, 2*i+1].copy()

# 使v1与y正交（边际相关归零）
proj_y = np.dot(v1, y_base) / (np.dot(y_base, y_base) + 1e-10)
v1_ortho = v1 - proj_y * y_base
v1_ortho = v1_ortho / (np.linalg.norm(v1_ortho) + 1e-10)

# 对v2同样处理
proj_y2 = np.dot(v2, y_base) / (np.dot(y_base, y_base) + 1e-10)
v2_ortho = v2 - proj_y2 * y_base
v2_ortho = v2_ortho / (np.linalg.norm(v2_ortho) + 1e-10)
```

### 5.2 为什么传统方法失效

1. **Lasso/ElasticNet/AdaptiveLasso**：依赖边际相关性（X[:,j]与y的相关性）进行变量选择，边际相关归零后无法识别真实信号
2. **AFL**：虽然使用Ridge第一阶段保留方向信息，但翻转和重建过程仍受边际相关归零影响
3. **UniLasso**：依赖单变量回归的显著性，同样受边际相关归零影响
4. **1-SE规则**：进一步将已经很弱的选择压缩到零

### 5.3 该场景的物理意义

绝对隐身陷阱模拟了现实中的"**隐性信号**"问题：
- 基因表达数据中的某些关键基因可能被其他相关基因的表达模式所掩盖
- 金融数据中某些重要因子与市场整体走势正交
- 工业检测中的某些缺陷特征被生产线噪声所掩盖

## 7. 结论

### 7.1 主要结论

1. **绝对隐身陷阱场景验证了边际相关性方法的局限性**：当真实信号的边际相关性被设计为接近零时，所有基于边际相关的变量选择方法都完全失效

2. **1-SE规则在该场景下过于保守**：导致本已困难的选择雪上加霜，4个模型选不出任何特征

3. **LassoCV_1SE相对最佳但仍远不够用**：F1仅0.0016，意味着平均1000次选择中只有约1.6次正确

### 7.2 实践建议

1. **在该场景下不应依赖自动CV参数选择**：1-SE规则过于保守
2. **如果必须进行变量选择**：可考虑使用更大的lambda搜索范围或不使用1-SE规则
3. **预测 vs 选择的权衡**：当预测MSE接近理论最小时（R²≈0），说明信号已被噪声淹没

### 7.3 改进方向

1. **开发对边际相关归零鲁棒的方法**：如基于条件独立性、互信息的方法
2. **考虑两阶段或多阶段策略**：先利用弱信号聚类，再在类内进行精细筛选
3. **引入领域知识**：在边际相关归零的情况下，可能需要利用其他形式的先验信息
