# AFL CV 1-SE Rule 完整信噪比实验报告

## 1. 实验概述

**实验目的**：评估 1-SE 法则对 AFL CV 过度选择问题的改进效果

**数据配置**：
- 相关性：AR(1), rho=0.8
- n_samples: 300
- n_features: 500
- n_nonzero: 20
- sigma 值：[0.1, 0.5, 1.0, 1.5, 2.0, 3.0]
- SNR 值：[10.0, 2.0, 1.0, 0.67, 0.5, 0.33]
- 重复次数：5 次/配置

**对比方案**：
1. **AFL-Fixed**：使用 Stage1 最优固定参数 (lambda_ridge=1.0, gamma=0.5)
2. **CV-Tuned (无1-SE)**：使用 min MSE 准则选择参数
3. **CV-1SE**：使用 1-SE 法则选择参数

## 6. 关键发现

### 6.1 1-SE 法则有效性

1. **显著减少过度选择**：n_selected 从 41.7 降至 24.5 (-41%)
2. **大幅提升 F1**：从 0.720 升至 0.874 (+21%)
3. **有效控制 FDR**：从 0.395 降至 0.194 (-51%)
4. **稳定性增强**：CV-1SE 的标准差远小于 CV-Tuned

### 6.2 与 Fixed 的差距

1. **差距极小**：F1 仅差 0.012 (1.4%)
2. **可接受的代价**：
   - n_selected 多选约 3.5 个特征
   - FDR 高约 0.016
3. **优于 Fixed 的情况**：在 sigma=3.0 (低 SNR) 时，CV-1SE 的 F1 (0.710) 高于 Fixed (0.693)

### 6.3 参数选择洞察

1. **gamma 偏好**：CV-1SE 选择 gamma≈0.53-0.70，而非 Fixed 的 0.5
2. **lambda_ridge 偏好**：低 SNR 时强烈偏好 2.0，可能导致过强收缩
3. **SNR 自适应**：参数选择随 SNR 降低而调整，显示一定的自适应能力

---

## 7. 结论

1. **1-SE 法则显著有效**：解决了 CV 过度选择问题，将 n_selected 降至接近真值

2. **与 Fixed 差距可接受**：
   - F1 差距仅 1.4%
   - n_selected 多约 17%
   - FDR 高约 9%

3. **推荐使用场景**：
   - 高 SNR 时：CV-1SE 几乎与 Fixed 等效
   - 无法获取先验知识时：CV-1SE 是 CV 的最佳选择
   - 低 SNR 时：仍优于纯 MSE 最优，但建议结合领域知识

4. **进一步优化方向**：
   - 调整搜索空间（特别是 lambda_ridge 上限）
   - 尝试 2-SE 法则进一步稀疏
   - 针对不同 SNR 使用自适应阈值

## 8. 代码修改

修改文件：`base.py` Stage 2 参数选择逻辑

```python
# 1-SE 法则：选择最稀疏的模型，其 MSE 在 1-SE 范围内
candidates_mask = mean_error <= threshold
if not np.any(candidates_mask):
    best_gamma_idx, best_alpha_idx = np.unravel_index(np.argmin(mean_error), ...)
else:
    masked_nselected = np.where(candidates_mask, mean_nselected, np.inf)
    best_flat_idx = np.argmin(masked_nselected)
    best_gamma_idx, best_alpha_idx = np.unravel_index(best_flat_idx, ...)
```
